#  Movie Recommendation System

A **Machine Learning-based Movie Recommendation System** that suggests similar movies using **content-based filtering** and **cosine similarity**. Built with Python and Streamlit for an interactive user experience.

---

##  Features

*  Search for any movie
*  Get top 5 similar movie recommendations
*  Fast and interactive UI
*  ML-based similarity model

---

##  Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## 🌐 Deployment

Deployed on Hugging Face Spaces
